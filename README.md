Bu proje, bir dermokozmetik markası olan Dermalyva için geliştirilmiş, yöneticilerin stratejik kararlar almasını kolaylaştıran web tabanlı bir Karar Destek Sistemi (KDS) ve Ürün Yönetim Paneli uygulamasıdır.

Proje, MVC (Model-View-Controller) mimarisine tam uyumlu olarak geliştirilmiş olup, dinamik simülasyonlar ve CRUD işlemleri içermektedir.
